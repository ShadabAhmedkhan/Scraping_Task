## 🚀 Features

- ✅ Scrapes **name, job title, office, email, and phone number**
- 🔁 Clicks **"Load more"** to reveal all professionals
- 🔐 Avoids duplicate scraping
- 💾 Saves output to `dentons_profiles.csv`
- ⚡ Uses `Selenium` for dynamic interaction
